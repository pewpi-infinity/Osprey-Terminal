# Osprey Terminal - Multi-Theme System

This directory contains configuration and state for the Osprey Terminal multi-theme system.

## Features
- 11 unique terminal themes
- Full MRW character system (Mario, Luigi, Cars, Mushrooms, Osprey, Rooster)
- Physics-based command execution with capacitor charging
- Joystick/gamepad navigation
- Theme-specific commands

## Initialization
The system initializes automatically when index.html loads all the required scripts.
